Nmap使用笔记

发表于 2018-05-26 | | 本文浏览数 次~
字数统计: 152

渗透的本质是信息搜集，而信息搜集整理为后续的情报跟进提供了强大的保证。

从上面那句话中，可见信息收集是多么的重要，因此，Nmap成为渗透中不可少的神器

Nmap在网络扫描中常见作用:

  1. 查看存活的主机
  2. 扫描目标主机开放端口
  3. 鉴定安全过滤机制
  4. 识别目标主机的操作系统
  5. 查看目标主机的服务banner
  6. 利用脚本扫描漏洞
